# Optics & Waves (High School) · Interactive

**Wave equation, interference, diffraction, EM spectrum**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 2 · High School

## What You'll Learn

- Wave properties: wavelength, frequency, speed
- Superposition and interference
- Diffraction and Huygens' principle
- Geometric optics: reflection and refraction
- The electromagnetic spectrum
